# Bakuage Stem Enhancer (Mac / Local)

目的: AI生成/分離後ステムで起きやすい以下をローカルで修復します。

- ステム化により局所的にトランジェントが喪失する
- シンバル(ライド含む)のサスティン過長、白ノイズ化、fizz/濁り
- 分離マスク由来の「水っぽさ」「ザラつき」
- ステム間の軽微なタイミング/極性ズレ

本ビルド(v0.4.0)は **2mix(分離前ミックス)が入力にある前提**で、
2mixをガイドに「欠損トランジェントの再配分」を行うのが中核です。

## 起動

1. ZIPを展開
2. `run.command` を右クリック→「開く」
3. ブラウザで `http://127.0.0.1:7860` が開きます

## 使い方

- Stems: 複数WAVをまとめて投入
- 2mix (recommended): 2mixを投入（推奨）
- Instrument mapping: `filename: label` 形式で上書き可能

出力は修復済みステムWAVとJSONレポートをZIPにして返します。

## MIDIについて

MIDIは将来拡張用です（オート生成MIDIの余計なノート混入に備え、
オーディオ側のオンセットと突き合わせて信頼度を調整する設計です）。
